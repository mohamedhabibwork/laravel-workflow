# Security Policy

## Supported Versions

Security fixes target the latest released major version.

## Reporting A Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Use GitHub private vulnerability reporting when available, or contact the maintainer at the email listed in `composer.json`.

Include:

- affected package version
- PHP and Laravel versions
- reproduction steps
- impact summary

We will acknowledge valid reports and coordinate a fix before public disclosure.
