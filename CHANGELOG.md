# Changelog

All notable changes to `laravel-workflow` will be documented in this file.

## Unreleased

- Added the Laravel workflow engine with versioned definitions, polymorphic subjects, action resolution, transitions, conditions, automation handlers, immutable history, and multi-tenancy support.
- Added PHP attribute workflow definitions and an attribute sync command.
- Added runtime controls for signals, updates, queries, cancellation, termination, retry, timers, delayed starts, timeouts, child workflows, continue-as-new, memo, and search attributes.
- Added durable activities with worker execution, retries, timeouts, asynchronous completion tokens, and Laravel event dispatch.
- Added workflow Artisan commands, full package documentation, tests, PHPStan coverage, and production GitHub workflows/templates.
